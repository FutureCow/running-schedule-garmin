Communicatie Next.js ↔ Python
Next.js /api/garmin routes proxyen calls naar de Python service via interne HTTP. Garmin credentials worden nooit direct aan de browser blootgesteld — alles loopt via de server. De Python service draait als kleine container op Railway (gratis tier volstaat voor klein gebruikersaantal).
